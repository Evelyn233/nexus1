# 用户专属界面说明

## 🎯 功能概述

创建了一个全新的用户专属页面（`/profile`），与 home 页面平级，提供完整的用户信息管理和创作统计功能。

## 📍 访问方式

### 1. 从主页进入
点击右上角的 `👤` 图标 → 直接跳转到个人主页

### 2. 直接访问
浏览器输入：`http://localhost:3000/profile`

```typescript
// 返回: { success, contents, total }
```

#### 3. 统计计算
```typescript
// 总图片数
const totalImages = contents.reduce((sum, content) => 
  sum + (content.imageCount || 0), 0
)

// 总创作次数
const totalContents = data.total

// 不同状态的内容统计
const draftCount = allContents.filter((c: any) => c.status === 'draft').length

  c.status === 'completed' || c.status === 'published'
).length
```

## 🚀 使用流程

```
  ↓

  ↓

  ↓
```

### 2. 编辑资料
```
点击"编辑资料"按钮

  ↓
```

```
Profile Page

  │   └─ 返回 { userInfo, userMetadata }
  │
  ├─ GET /api/user/generated-content?limit=5&offset=0
  │   └─ 返回 { success, contents, total }
  │
  └─ 计算统计数据
      ├─ totalImages = Σ(content.imageCount)
      ├─ totalContents = data.total
      ├─ draftContents = contents.filter(status === 'draft').length
      ├─ privateContents = contents.filter(status === 'private').length
      ├─ publishedContents = contents.filter(status === 'completed' || 'published').length
      ├─ joinDate = userInfo.createdAt
      └─ lastActive = new Date()
```

## 🎯 与 Home 页面的区别

| 特性 | Home 页面 | Profile 页面 |
|------|----------|-------------|
| 主要功能 | 开始新的创作 | 查看个人信息和统计 |
| 内容展示 | 静态示例卡片 | 用户真实数据 |
| 导航位置 | 入口页面 | 二级页面 |
| 用户信息 | 弹窗显示（简略） | 独立页面（完整） |
| 统计数据 | 无 | 详细统计（含状态分类） |
| 最近创作 | 无 | 最近5条 |
| 内容状态 | 无 | 草稿/私密/已发布 |
