# 快速生成功能优化说明

## 🎯 优化目标

让"快速生成"示例更加**多元化**和**贴近真实场景**，涵盖用户的各种真实输入情况。

## 📊 优化前后对比

### 优化前（6个示例）
```
- 我今天上班很伤心
- 感觉被老板忽视了
- 中国缺乏高端杂志市场
- AI会取代人类工作
- 今天在公司开会
- 周末和朋友聚餐
```
**问题**：场景类型太少，不够多元化

## 🎨 设计理念

### 1. 场景多元化
- 涵盖工作、生活、社交、情感等各个方面
- 从正面到负面，从日常到特殊
- 符合真实用户的输入习惯

### 2. 情绪覆盖全面
- 😊 开心：惊喜礼物、升职、学会新技能
- 😢 伤心：被忽视、面试被拒、吵架
- 😰 压力：deadline、纠结、迷茫
- 😌 放松：追剧、跑步、看书

### 3. 生活场景真实
- 职场：开会、加薪、入职、完成项目
- 社交：聚餐、逛街、团圆饭、喝咖啡
- 独处：追剧、看书、做饭、跑步
- 成长：升职、学习、创业、健身

### 4. 适合不同用户
- **上班族**：工作压力、职场关系、升职加薪
- **创业者**：项目挑战、融资、成长机会
- **学生**：学习成长、社交困扰
- **普通人**：日常生活、情感困扰、自我反思

## 📝 使用指南

### 给用户的建议

**如何选择示例？**

1. **按情绪选**
   - 开心时 → 选择成就/社交类
   - 难过时 → 选择困境/情绪类
   - 迷茫时 → 选择思考类

2. **按场景选**
   - 在工作中 → 选择工作类
   - 在社交中 → 选择社交类
   - 独处时 → 选择生活日常类

3. **按目的选**
   - 想表达观点 → 选择观点类
   - 想记录生活 → 选择生活/社交类
   - 想自我反思 → 选择思考类

**选择后可以修改吗？**

✅ 可以！点击示例后会自动填入输入框，你可以修改后再发送。
