# 聊天历史记录功能文档

## 功能概述

实现了类似 ChatGPT 的聊天历史记录和图片生成记录存储功能，用户可以方便地查看、管理和恢复历史对话。

## 主要功能

### 1. 历史记录侧边栏 📁

- **位置**: 所有主要页面（主页、聊天页面）的左上角历史记录按钮
- **功能**:
  - 查看所有历史对话记录
  - 查看所有生成的图片记录
  - 支持搜索和过滤
  - 显示创建时间和对话摘要

### 2. 对话记录管理 💬

#### 自动保存
- 用户每次对话自动保存到本地存储
- 包含完整的对话内容、问题和答案
- 支持断点续聊，不会丢失历史记录

#### 对话列表
- 显示所有历史对话，按时间倒序排列
- 每个对话显示：
  - 标题（取自初始提示的前50个字符）
  - 创建时间（今天、昨天、X天前）
  - 对话摘要预览

#### 对话操作
- **点击对话**: 恢复历史对话，继续聊天
- **删除对话**: 支持删除不需要的历史记录
- **新对话**: 一键创建新的对话

### 3. 图片生成记录 🖼️

#### 图片列表
- 以网格形式展示所有生成的图片
- 每张图片显示：
  - 缩略图预览
  - 生成提示词
  - 生成时间

#### 图片操作
- **查看大图**: 点击图片查看完整尺寸
- **删除图片**: 删除不需要的图片记录

### 4. 新对话功能 ✨

- 侧边栏顶部的"新对话"按钮
- 点击后返回主页，可以开始新的对话
- 不影响现有的历史记录

## 技术实现

### 后端 API

#### 1. 聊天会话 API (`/api/chat-sessions`)

**GET** - 获取所有会话列表
```typescript
Response: {
  sessions: Array<{
    sessionId: string
    title: string
    messages: ChatMessage[]
    initialPrompt: string
    answers: string[]
    questions: string[]
    createdAt: string
    updatedAt: string
  }>
}
```

**POST** - 保存或更新会话
```typescript
Request: {
  sessionId: string
  title: string
  messages: ChatMessage[]
  initialPrompt: string
  answers: string[]
  questions: string[]
}
```

**DELETE** - 删除指定会话
```typescript
Query: sessionId=xxx
```

#### 2. 图片记录 API (`/api/saved-images`)

已有接口，返回所有保存的图片列表

### 前端组件

#### ChatHistorySidebar 组件

**位置**: `components/ChatHistorySidebar.tsx`

**Props**:
- `isOpen: boolean` - 控制侧边栏显示/隐藏
- `onClose: () => void` - 关闭侧边栏的回调
- `onSessionSelect: (session) => void` - 选择历史会话的回调

**特性**:
- 响应式设计，适配移动端和桌面端
- 平滑的动画过渡效果
- 支持标签切换（对话记录 / 图片记录）
- 自动加载和刷新数据

### 数据存储

**存储位置**: `public/chat-sessions/`

**文件格式**: JSON
```
session_{timestamp}.json
```

**数据结构**:
```json
{
  "sessionId": "session_1696995123456",
  "title": "我明天面试穿什么",

  "updatedAt": "2023-10-10T12:30:00.000Z"

```

## 使用指南

### 1. 打开历史记录

在任意页面点击左上角的历史记录图标（时钟图标）

### 2. 查看对话记录

1. 默认显示"对话记录"标签
2. 向下滚动查看所有历史对话
3. 点击任意对话卡片恢复该对话

### 3. 查看图片记录

1. 点击"图片记录"标签
2. 以网格形式浏览所有生成的图片
3. 点击图片查看大图

### 4. 删除记录

- **对话记录**: 悬停在对话卡片上，点击右上角的删除图标
- **图片记录**: 悬停在图片上，点击中央的删除按钮

### 5. 开始新对话

1. 在历史记录侧边栏顶部点击"+ 新对话"按钮
2. 或直接关闭侧边栏，在主页输入新的提示

## 用户界面集成

### 主页 (app/home/page.tsx)
- 左上角添加历史记录按钮
- 点击打开历史记录侧边栏
- 选择历史会话后跳转到聊天页面

### 聊天页面 (app/chat-new/page_new.tsx)
- 左上角历史记录按钮
- 自动保存当前对话
- 支持加载历史会话

### 生成页面 (app/generate/page.tsx)
- 可以访问图片历史记录
- 查看之前生成的所有图片

## 性能优化

1. **延迟保存**: 对话内容变化后延迟1秒保存，避免频繁写入
2. **按需加载**: 只在打开侧边栏时加载历史记录
3. **缓存策略**: 图片使用本地路径，无需重复下载

## 注意事项

1. 所有数据存储在本地 `public/chat-sessions/` 目录
2. 删除会话不可恢复，请谨慎操作
3. 建议定期备份重要对话记录
4. 图片存储在 `public/generated-images/` 目录

## 故障排除

### 历史记录不显示
1. 检查 `public/chat-sessions/` 目录是否存在
2. 检查浏览器控制台是否有错误信息
3. 尝试刷新页面重新加载

### 对话无法恢复
1. 确认会话文件存在且格式正确
2. 检查sessionId是否正确
3. 查看浏览器控制台错误日志

### 删除失败
1. 确认有文件写入权限
2. 检查API路由是否正常工作
3. 查看服务器日志
