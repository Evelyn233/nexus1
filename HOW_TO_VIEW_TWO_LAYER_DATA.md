# 如何查看两层数据结构（表意识和潜意识）

## 快速开始

### 方法1：通过网页查看（推荐）

1. **启动项目**
   ```bash
   npm run dev
   ```

2. **登录系统**
   - 访问 `http://localhost:3000`
   - 登录你的账号

3. **查看两层数据**
   - 访问 `http://localhost:3000/two-layer-data`
   - 或在首页添加链接到这个页面

4. **切换数据层级**
   - 点击"第一层：表意识"查看用户真实说的、做的
   - 点击"第二层：潜意识"查看AI分析的深层模式

### 方法2：通过API查看

1. **调用API**
   ```bash
   curl http://localhost:3000/api/user/two-layer-data \
     -H "Cookie: next-auth.session-token=YOUR_SESSION_TOKEN"
   ```

2. **返回数据结构**
   ```json
   {
     "userId": "用户ID",
     "userName": "用户名",
     "consciousLayer": {
       "description": "第一层：表意识",
       "data": {
         "userProfile": { ... },
         "conversations": [ ... ],
         "rawInputs": [ ... ],
         "mentionedKeywords": [ ... ]
       }
     },
     "subconsciousLayer": {
       "description": "第二层：潜意识",
       "data": {
         "coreTraits": [ ... ],
         "emotionalPattern": [ ... ],
         "behaviorPatterns": [ ... ],
         ...
       }
     }
   }
   ```

## 数据说明

### 第一层：表意识（Conscious Layer）

这一层存储的是**用户真实说的、做的、表现的**：

#### 1. 用户自我描述
- 性格描述（用户自己填写的）
- MBTI类型
- 自我特质
- 兴趣爱好
- 目标愿景

#### 2. 对话记录
- 最近的对话记录
- 用户的初始输入
- 用户的回答
- 对话时间

#### 3. 原始输入
- 用户历史输入记录
- 原汁原味的用户表达

#### 4. 提到的关键词
- 从对话中提取的关键词
- 用户经常提到的词汇

### 第二层：潜意识（Subconscious Layer）

这一层存储的是**AI分析的深层心理模式**：

#### 1. 核心性格特征
- AI分析得出的核心特质
- 基于用户行为和对话的推测

#### 2. 情感模式
- 用户的情绪反应模式
- 情感表达方式

#### 3. 行为模式
- 用户的行为习惯
- 应对方式

#### 4. 压力反应
- 面对压力时的反应
- 应对策略

#### 5. 人际关系
- 人际优势
- 人际挑战
- 社交能量模式

#### 6. 生活方式
- 审美偏好
- 生活爱好
- 活动偏好
- 时尚风格

#### 7. 地点偏好
- 常去的地点
- 喜欢的场所类型

#### 8. 职业和关系
- 职业天赋
- 感情模式
- 人生哲学

#### 9. 对话洞察
- 从对话中发现的深层洞察
- AI的综合分析

## 数据来源

### 表意识数据来源
1. **用户填写** - 注册时填写的基本信息
2. **用户对话** - 每次对话的记录
3. **用户输入** - 用户键入的原始内容

### 潜意识数据来源
1. **AI分析** - 基于用户对话的AI分析
2. **行为学习** - 从用户行为中学习
3. **模式识别** - 识别用户的深层模式

## 优先级规则

根据 `docs/USER_DATA_PRIORITY.md`：

### 第1优先级：用户自我认知（100%权重）
- 用户自己说的性格描述
- 用户填写的MBTI
- 用户的自我特质

### 第2优先级：聊天记录（100%权重）
- 用户每次回答
- 用户表达的想法和感受
- 用户描述的经历

### 第3优先级：AI分析（辅助解释，20%权重）
- AI推测的性格特征
- AI分析的行为模式
- AI总结的洞察

**注意：** 第1、2层是事实，第3层是推测和辅助解释。

## 数据更新机制

### 表意识数据
- 每次对话后立即更新
- 用户修改个人信息时更新
- 实时反映用户最新状态

### 潜意识数据
- 异步分析更新
- 累积式存储（不覆盖旧数据）
- AI模型升级时可重新分析

## 使用场景

### 1. 了解自己
查看AI如何理解你的性格和行为模式。

### 2. 验证分析
对比第一层（你说的）和第二层（AI推测的），验证AI分析是否准确。

### 3. 发现盲点
第二层可能揭示你自己没有意识到的行为模式。

### 4. 追踪变化
随着时间推移，查看自己的性格和行为是否有变化。

## 常见问题

### Q: 为什么我的潜意识数据很少？
A: 潜意识数据需要通过多次对话积累。多进行对话，AI会逐渐学习和分析。

### Q: AI分析的准确吗？
A: AI分析基于你的对话内容，准确性取决于你提供的信息量和质量。建议作为参考，而非绝对真理。

### Q: 可以删除某些数据吗？
A: 目前暂不支持选择性删除。如需完全删除，请联系管理员。

### Q: 数据会被用于其他用途吗？
A: 不会。数据仅用于为你提供个性化服务，不会分享给他人。
