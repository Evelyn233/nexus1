## 📰 杂志封面设计规则

### 核心原则：内容决定设计

**不是这样（错误）：**
```
❌ 套用固定模板
❌ 使用通用标题（"我的故事"）
❌ 模糊的配色（"温暖色调"）
❌ 笼统的画风（"现代风格"）
```

**而是这样（正确）：**
```
✅ 分析故事类型 → 确定风格
✅ 提取核心主题 → 生成标题
✅ 分析情感走向 → 确定配色
✅ 理解故事气质 → 设计画风
```

### 设计流程

```
故事内容
    ↓
分析类型和情感
    ↓
┌─────────────────────────┐
│ 挣扎突破类？            │
│ → 戏剧性、强对比        │
│ → 深色→亮色渐变         │
│ → 电影感光影            │
│ → 标题：逆境重生、破茧  │
├─────────────────────────┤
│ 成长蜕变类？            │
│ → 温暖励志              │
│ → 温暖渐变色            │
│ → 柔和希望感            │
│ → 标题：向光而行、绽放  │
├─────────────────────────┤
│ 情感治愈类？            │
│ → 文艺细腻              │
│ → 柔和暖色              │
│ → 温柔诗意              │
│ → 标题：和解、归途      │
├─────────────────────────┤
│ 都市职场类？            │
│ → 现代简约              │
│ → 高级灰、黑白          │
│ → 专业利落              │
│ → 标题：选择、平衡      │
└─────────────────────────┘
    ↓
生成专属封面设计
```

### 设计示例

#### 案例A：高考失利 → 出国留学

**故事分析：**
- 类型：挣扎突破
- 情感：痛苦失望 → 希望新生
- 冲突：传统教育压力 vs 追求自我

**封面设计：**
```
大标题：逆境重生
小标题：从高考失利到海外求学的勇敢抉择

风格：戏剧性、强情感对比
配色：深蓝沉重感 渐变到 暖橙希望色
画风：电影级戏剧光影，从暗到亮的情绪转变

构图：
19岁女性站在悉尼大学门前
回望身后的路（象征回望失败）
眼神坚定向前（象征新生）
背景从暗到亮的光线渐变
左侧深蓝暗部，右侧暖橙亮部
```

#### 案例B：都市压力 → 找到平静

**故事分析：**
- 类型：都市治愈
- 情感：焦虑压抑 → 平静释放
- 冲突：快节奏压力 vs 内心需要平静

**封面设计：**
```
大标题：呼吸
小标题：在快节奏都市中寻找内心平静

风格：现代简约 + 温暖文艺
配色：冷灰都市感 融合 米色温暖治愈
画风：都市剪影 + 柔和温暖光线

构图：
都市高楼剪影背景（冷灰色）
人物在温暖的角落（咖啡厅/窗边）
柔和光线洒在身上（米色、淡橙）
放松姿态，闭眼或远眺
从冷硬都市到温暖角落的视觉对比
```

## 封面生成的特点

### 🎯 完全根据内容设计

**分析故事 → 决定设计：**
1. 故事类型 → 封面风格
2. 情感走向 → 配色方案
3. 故事气质 → 画风选择
4. 核心主题 → 大标题
5. 核心情节 → 小标题
6. 最戏剧moment → 封面构图

### 📊 设计规则表

| 故事类型 | 风格 | 配色 | 画风 | 标题示例 |
|---------|------|------|------|---------|
| 挣扎突破 | 戏剧性 | 深色→亮色 | 电影感光影 | 逆境重生、破茧 |
| 成长蜕变 | 温暖励志 | 温暖渐变 | 柔和希望 | 向光而行、绽放 |
| 情感治愈 | 文艺细腻 | 柔和暖色 | 温柔诗意 | 和解、归途 |
| 都市职场 | 现代简约 | 高级灰黑白 | 专业利落 | 选择、平衡 |
| 追梦突破 | 动感明亮 | 明亮向上 | 清晰有力 | 追光者、远航 |

### ✨ 独特性保证

每个故事的封面都是独特的，因为：
1. 标题从故事中提炼（不用模板）
2. 风格根据类型确定（不固定）
3. 配色根据情感确定（动态选择）
4. 画风根据气质确定（具体描述）
5. 构图根据moment确定（基于内容）
