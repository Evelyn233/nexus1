# 测试指南

## 🚀 快速测试步骤

### 1. 启动应用
```bash
cd magazine
npm install
npm run dev
```

### 2. 测试完整流程

#### 步骤1: 主页输入
1. 打开 `http://localhost:3000`
2. 在底部输入框输入："我明天面试穿什么"
3. 点击"发送"按钮

#### 步骤2: 智能问答
1. 自动跳转到对话页面 (`/chat`)
2. 看到AI生成的5个深度问题
3. 逐一回答问题（或点击"跳过"直接生图）

#### 步骤3: 图片生成
1. 自动跳转到生图页面 (`/generate`)
2. 等待3-5秒图片生成完成
3. 看到3张生成的图片
4. 查看本地保存提示信息

#### 步骤4: 查看本地图片
1. 点击主页右上角的🖼️图标
2. 进入图片画廊 (`/gallery`)
3. 查看所有本地保存的图片
4. 测试下载、删除功能

## 🐛 常见问题

### 1. 图片不显示
**原因**: 可能是网络问题或API限制
**解决**: 检查控制台错误信息，本地模拟API应该能正常工作

### 2. 对话页面卡住
**原因**: API响应超时
**解决**: 点击"跳过"按钮直接进入生图页面

### 3. 本地图片保存失败
**原因**: 文件权限问题
**解决**: 确保 `public/generated-images` 目录有写入权限

### 4. 画廊页面空白
**原因**: 没有生成过图片
**解决**: 先完成一次完整的生图流程

## 🔧 调试技巧

### 查看控制台日志
打开浏览器开发者工具，查看：
- API调用日志
- 错误信息
- 图片保存状态

### 检查网络请求
在Network标签页查看：
- `/api/doubao-chat` - 对话API
- `/api/generate-image` - 生图API
- `/api/save-image` - 保存图片API

### 查看本地文件
检查 `public/generated-images` 目录：
- 图片文件 (.jpg)
- 元数据文件 (.json)
