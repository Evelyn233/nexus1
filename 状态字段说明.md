# UserGeneratedContent 状态字段说明

## 📊 状态定义

在 `UserGeneratedContent` 表中，`status` 字段用于跟踪用户创作内容的不同阶段和可见性：

### 状态类型

```typescript
type ContentStatus = 
  | 'draft'      // 草稿 - 未完成的创作
  | 'processing' // 处理中 - 正在生成图片/内容
  | 'completed'  // 已完成 - 创作完成（默认状态）
  | 'failed'     // 失败 - 生成失败
  | 'private'    // 私密 - 仅自己可见
  | 'published'  // 已发布 - 公开可见
```

## 🔍 状态统计逻辑

### 前端计算

```typescript
// 统计不同状态的内容
const allContents = contentsData.contents || []

// 🟡 已创作（草稿）
const draftCount = allContents.filter((c: any) => 
  c.status === 'draft'
).length

// 🔒 锁着的（私密）
const privateCount = allContents.filter((c: any) => 
  c.status === 'private'
).length

// 🟢 已发布（公开）
const publishedCount = allContents.filter((c: any) => 
  c.status === 'completed' || c.status === 'published'
).length
```

## 📝 状态流转图

```
用户开始创作
  ↓
[draft] 草稿
  ↓
用户继续创作并提交
  ↓
[processing] 处理中
  ↓
  ├─ 成功 → [completed] 已完成（默认）
  │          ↓
  │          ├─ 用户设置为私密 → [private] 私密
  │          └─ 用户设置为公开 → [published] 已发布
  │
  └─ 失败 → [failed] 失败
```

## 🎯 状态变更场景

### 1. 用户创建新内容
```typescript
// 默认状态：completed（已完成）
const newContent = {
  userId: user.id,
  initialPrompt: "...",
  status: "completed"  // 默认值
```

```typescript

  where: { id: contentId },
  data: { status: "draft" }
```

## 🔐 权限控制

### 内容可见性规则

| 状态 | 本人可见 | 他人可见 | 出现在公开列表 |
|------|---------|---------|---------------|
| draft | ✅ | ❌ | ❌ |
| processing | ✅ | ❌ | ❌ |
| completed | ✅ | ❌ | ❌ |
| failed | ✅ | ❌ | ❌ |
| private | ✅ | ❌ | ❌ |
| published | ✅ | ✅ | ✅ |

### API 权限检查

```typescript
// 获取内容详情
async function getContentById(contentId: string, userId: string) {
  const content = await prisma.userGeneratedContent.findUnique({
    where: { id: contentId }
  })
  
  // 权限检查
  if (content.status === 'published') {
    // 已发布内容，所有人可见
    return content
  } else {
    // 其他状态，仅本人可见
    if (content.userId === userId) {
      return content
    } else {
      throw new Error('无权访问此内容')
    }
  }
}
```

## 📋 总结

- ✅ 默认状态：`completed`（已完成）
- ✅ 统计维度：已创作（draft）、锁着的（private）、已发布（completed/published）
- ✅ 状态流转：草稿 → 处理中 → 已完成 → 私密/已发布
- ✅ 权限控制：仅 `published` 状态对外公开
