# 搜索功能说明

## 🔍 功能概述

为聊天记录侧边栏和用户Profile页面添加了实时搜索功能，方便用户快速查找历史创作内容。

## 📍 搜索框位置

### 1. 聊天记录侧边栏（ChatHistorySidebar）

```
┌─────────────────────────┐
│ 历史记录           ✕   │
├─────────────────────────┤
│    + 新对话             │
├─────────────────────────┤
│ 🔍 [搜索创作内容...]    │ ← 搜索框
├─────────────────────────┤
│ ✨ 故事记录             │
├─────────────────────────┤
│ [创作列表...]           │
└─────────────────────────┘
```

**位置**: 在"新对话"按钮下方，故事记录标签上方

### 2. Profile页面 - 最近创作区域

```
┌─────────────────────────────────┐
│ ✨ 最近创作      查看全部 →     │
├─────────────────────────────────┤
│ 🔍 [搜索你的创作...]            │ ← 搜索框
├─────────────────────────────────┤
│ [创作列表...]                   │
└─────────────────────────────────┘
```

**位置**: 在"最近创作"标题下方，创作列表上方

## 🎨 UI 设计

### 搜索框样式

```tsx
<div className="relative">
  <Search className="absolute left-3 top-1/2 -translate-y-1/2 w-4 h-4 text-gray-400" />
  <input
    type="text"
    placeholder="搜索创作内容..."
    value={searchQuery}
    onChange={(e) => setSearchQuery(e.target.value)}
    className="w-full pl-10 pr-10 py-2 border border-gray-200 rounded-lg text-sm focus:outline-none focus:ring-2 focus:ring-purple-500 focus:border-transparent"
  />
  {searchQuery && (
    <button onClick={() => setSearchQuery('')}>
      <X className="w-4 h-4" />
    </button>
  )}
</div>
```

**特点**：
- 左侧搜索图标（🔍）
- 右侧清除按钮（X，仅在有搜索内容时显示）
- 圆角边框
- 聚焦时显示彩色边框（teal/purple）
- 响应式设计

## 📊 搜索特性

### 1. 实时搜索
- ✅ 无需点击搜索按钮
- ✅ 输入即时过滤结果
- ✅ 性能优化（前端过滤）

### 2. 模糊匹配
- ✅ 不区分大小写
- ✅ 匹配部分文本
- ✅ 支持中英文搜索

### 3. 多字段搜索
- ✅ 搜索用户输入（initialPrompt）
- ✅ 搜索故事内容（storyNarrative）

### 5. 快速清除
- ✅ 输入框右侧 X 按钮

- ✅ 一键恢复全部结果

## 🔍 搜索算法

### 当前实现（前端过滤）

```typescript
// 简单的 includes 匹配
const query = searchQuery.toLowerCase()
return (
  content.initialPrompt?.toLowerCase().includes(query) ||
  content.storyNarrative?.toLowerCase().includes(query)
)
```

**优点**：
- ✅ 实时响应快
- ✅ 无需后端支持
- ✅ 实现简单

**适用场景**：
- ✅ 数据量较小（< 100条）
- ✅ 已加载到前端的数据
