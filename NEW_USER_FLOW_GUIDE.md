# 新用户完整流程指南

## 🎯 完整流程

### 步骤1：注册账号（二选一）

#### 方式A：手机号注册（推荐）✅
```
访问: http://localhost:3000/auth/signup-phone

1. 输入手机号（如：13800138000）
2. 点击"发送验证码"
3. 查看验证码（开发环境会显示）
4. 输入验证码
5. 设置密码
6. 点击"注册"

✅ 账号创建完成
```

#### 方式B：邮箱注册
```
访问: http://localhost:3000/auth/signup

1. 输入姓名（可选）
2. 输入邮箱
3. 设置密码
4. 点击"注册"

✅ 账号创建完成
```

### 步骤2：填写详细信息（必须完成！）

```
自动跳转: http://localhost:3000/user-info

填写信息:
├─ 姓名 *
├─ 性别 * (男/女)
├─ 生日 * (年月日)
├─ 身高 *
├─ 体重 *
├─ 所在地 *
├─ 性格描述 * (如：INTJ 理性 创业者)
└─ 头发长度 * (女性必填)

⚠️⚠️⚠️ 重要：必须点击"提交"按钮！
⚠️⚠️⚠️ 不要关闭页面！
⚠️⚠️⚠️ 等待保存成功提示！

✅ 详细信息保存完成
```

### 步骤3：开始聊天

```
访问: http://localhost:3000/chat-new

功能:
- 输入今天发生的事
- 回答系统问题
- 积累对话数据

数据存储:
✅ 聊天记录 → ChatSession表
✅ 对话内容 → 用于后续分析
✅ 情绪检测 → 用于心理剧生成
```

### 步骤4：生成内容

```
访问: http://localhost:3000/generate

流程:
1. 系统读取你的:
   - 基本信息（性别、年龄、地点）
   - 聊天记录（你说了什么）
   - 两层数据（表意识+潜意识）

2. 生成内容:
   - 场景描述
   - 故事叙述
   - 🎭 心理剧场景（如果检测到情绪）
   - 图像提示词

3. 生成图片:
   - 基于你的真实经历
   - 包含你的外貌特征
   - 体现你的情绪状态

✅ 内容生成完成
```

## 🚨 注意事项

### 为什么昨天的用户没保存？

**可能原因：**
1. 在 Step 2（填写详细信息）时关闭了页面
2. 填写了但没点"提交"
3. 点了提交但网络失败
4. Session过期导致保存失败

**后果：**
- 账号创建了（User表有记录）
- 但没有详细信息（gender等为NULL）
- 系统可能清理了这种不完整账号

### 如何避免数据丢失？

✅ **关键点：**
1. 注册后**立即填写详细信息**
2. **必须点击"提交"按钮**
3. **等待"保存成功"提示**
4. **看到跳转再关闭页面**

## 📱 手机号注册的优势

```
对比：

邮箱注册:
/signup → 创建账号 → /user-info → 填写资料
        (可能中断)           (可能中断)

手机号注册:
/signup-phone → 验证手机 → 创建账号 ✅ → /user-info → 填写资料
     (验证码验证)       (一步完成)              (可能中断)

优势：
✅ 验证码验证，更安全
✅ 手机号即账号，更简单
✅ 减少一步流程
```
